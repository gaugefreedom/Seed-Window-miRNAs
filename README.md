# Seed-Window-miRNAs
Seed-Window Algebraic Instability in Dementia-Associated miRNAs

This repository contains the code and pilot outputs for the manuscript:
“Seed-Window Algebraic Instability in Dementia-Associated miRNAs: A Pilot Study.”

Archived software release used for the manuscript:
- Zenodo DOI: https://doi.org/10.5281/zenodo.19489106

Contents:
- algebraic profiling code
- pilot cohort input file
- window-level results
- summary tables and figures used in the manuscript
